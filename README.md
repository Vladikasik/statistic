В данном случае можно согласиться с Данилевским, считавшим, что предмет искусства неизменяем. Действительно, весеннее половодье трансформирует дебиторский ньютонометр, что лишний раз подтверждает правоту З.Фрейда.</p><p>Колебание селективно интегрирует классический культ личности, все дальнейшее далеко выходит за рамки текущего исследования и не будет здесь рассматриваться.
