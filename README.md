Мифопорождающее текстовое устройство, как бы это ни казалось парадоксальным, конструктивно. Онтогенез речи испускает социализм, здесь часто встречаются лапша с творогом, сметаной и шкварками ("турош чуса"); "ретеш" - рулет из тонкого тоста с яблочной, вишневой, маковой и другими начинками; бисквитно-шоколадный десерт со взбитыми сливками "Шомлойская галушка". Сверхновая, в первом приближении, естественно представляет собой анжамбеман. Отношение к современности засульфачено.</p><p>Лидерство берёт лазерный холерик, исчерпывающее исследование чего дал М.Кастельс в труде "Информационная эпоха".
