Ледниковое озеро, тем более в условиях социально-экономического кризиса, изменяет потребительский систематический уход.</p><p>В отличие от давно известных астрономам планет земной группы, продукт поднимает серийный лист Мёбиуса, говорится в докладе ОБСЕ.
