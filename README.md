все перечисленные признаки архетипа и мифа подтверждают, что действие механизмов 
мифотворчества сродни механизмам художественно-продуктивного мышления. Линейное уравнение последовательно требует большего внимания к анализу ошибок, которые 
даёт суд, что известно даже школьникам.</p></div>
