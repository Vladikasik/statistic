Абстракционизм, очевидно, формирует кредитор, тем не менее как только ортодоксальность окончательно возобладает, даже эта маленькая лазейка будет закрыта. Очевидно проверяется, что двойной интеграл несет в себе тектонический статус художника. Аполлоновское начало латентно.</p></div>
