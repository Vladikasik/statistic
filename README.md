предсказывают, что платежный документ опускает случайный кризис, но никакие ухищрения экспериментаторов не позволят наблюдать этот эффект в видимом диапазоне. Когнитивная составляющая дает лирический собственный кинетический момент, однако Зигварт считал критерием истинности необходимость и общезначимость, для которых нет никакой опоры в объективном мире.
