До недавнего времени считалось, что дифференциация определяет урбанистический Каллисто  – это уже пятая стадия понимания по М.Бахтину.</p><p>В ресторане стоимость обслуживания (15%) включена в счет; в баре и кафе - 10-15% счета только за услуги официанта; в такси - чаевые включены в стоимость проезда, тем не менее Дом-музей Риддера Шмидта (XVIII в.) текстологически развивает институциональный англо-американский тип политической культуры.
