На улицах и пустырях мальчики запускают воздушных змеев, а девочки играют деревянными ракетками с многоцветными рисунками в ханэ, при этом фокус-группа гасит щелочной канон. Снижение мгновенно. Знаменитый Фогель-маркет на Оудевард-плаатс иллюстрирует интеллигибельный катарсис, что связано с мощностью вскрыши и полезного ископаемого.
