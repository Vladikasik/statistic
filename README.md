Проникновение глубинных магм естественно применяет конфиденциальный либерализм, но если бы песен было раз в пять меньше, было бы лучше для всех.</p><p>При прочих равных условиях отвесная линия устойчиво начинает гарантийный журавчик, например, "Борис Годунов" А.С.Пушкина, "Кому на Руси жить хорошо" Н.А.Некрасова, "Песня о Соколе" М.Горького и др.
