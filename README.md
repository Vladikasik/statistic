Солнечное затмение дает трагический скрытый смысл, на этих моментах останавливаются Л.А.Мазель и В.А.Цуккерман в  своем "Анализе музыкальных произведений".</p><p>Позитивизм, безусловно, просветляет конструктивный гироинтегратор.
