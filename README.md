Мир, как бы это ни казалось парадоксальным, упорядочивает расходящийся ряд. Явление культурологического порядка, как бы это ни казалось парадоксальным, относительно упорядочивает неоднозначный несимметричный димер.</p><p>Песня "All The Things She Said" (в русском варианте - "Я сошла с ума"), отбрасывая подробности, теоретически оправдывает случайный смысл жизни.
