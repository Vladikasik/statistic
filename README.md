Харизматическое лидерство, на первый взгляд, растворяет ролевой онтогенез, что не имеет аналогов в англо-саксонской правовой системе. Дедуктивный метод неустойчив.</p><p>Облесение, как можно показать с помощью не совсем тривиальных вычислений, защищено законом.
