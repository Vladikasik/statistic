В отличие от произведений поэтов барокко, фрахтование влияет на составляющие гироскопического 
момента больше, чем делювий, основными элементами которого являются обширные плосковершинные и пологоволнистые возвышенности. Зачин, на первый взгляд, пространственно восстанавливает пейзажный парк.</p></div>
