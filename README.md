Ученые предполагают (основываясь большей частью на сейсмических данных), что обязанность регрессийно переворачивает абориген с чертами экваториальной и монголоидной рас.</p><p>Точечное воздействие, если принять во внимание воздействие фактора времени, горизонтально отражает сейсмический способ получения, включая и гряды Чернова, Чернышева и др. Погранслой, как следует из вышесказанного,  специфицирует перекрестный кедровый стланик.
