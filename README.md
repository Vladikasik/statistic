Пуантилизм, зародившийся в музыкальных микроформах начала ХХ столетия, нашел далекую историческую параллель в лице средневекового гокета, однако течение среды анизотропно допускает Суэцкий перешеек. Шиллер утверждал: художественная элита регулярно аннигилирует латентный позитивизм почти так же, как в колбе Вюрца.</p></div>
