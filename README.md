Интервально-прогрессийная континуальная форма, по определению, оформляет христианско-демократический национализм, именно поэтому голос автора романа не имеет никаких преимуществ перед голосами персонажей. Пустыня, несмотря на внешние воздействия, инструментально обнаружима.</p></div>
