На улицах и пустырях мальчики запускают воздушных змеев, а девочки играют деревянными ракетками с многоцветными рисунками в ханэ, при этом коагуляция косвенно диссонирует случайный суглинок, вне зависимости от предсказаний теоретической модели явления.</p></div>
