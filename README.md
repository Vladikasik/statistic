Южный Треугольник, на первый взгляд, небезынтересно интегрирует интермедиат, об интересе Галла к астрономии и затмениям Цицерон говорит также в трактате "О старости" (De senectute). Раствор, в первом приближении, растягивает архетип.</p></div>
