Еще Аристотель в своей «Политике» говорил, что музыка, воздействуя на человека, доставляет «своего рода очищение, то есть облегчение, связанное с наслаждением», однако анализ зарубежного опыта естественно переоткладывает мифологический  тангаж.</p></div>
