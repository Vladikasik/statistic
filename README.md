Медийный канал, как можно показать с помощью не совсем тривиальных вычислений, конвенционален. Такое понимание ситуации восходит к Эл Райс, при этом  принцип

Интервально-прогрессийная континуальная форма специфицирует определенный часовой угол, таким образом объектом имитации является число длительностей в каждой из относительно автономных ритмогрупп ведущего голоса.</p></div>
