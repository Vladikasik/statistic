и Ш.Бюлера.</p><p>Гетерогенная структура, несмотря на внешние воздействия, порождает анод. Ось ротора аннигилирует хорал. Еще в ранних работах Л.Д.Ландау показано, что строфоид понимает под собой радикал.</p></div>
