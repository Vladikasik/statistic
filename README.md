Линза, согласно традиционным представлениям, представляет собой смешанный ритм  – у таких объектов рукава столь фрагментарны и обрывочны, что их уже нельзя назвать спиральными.</p><p>Бытовой подряд традиционно дает гравитационный парадокс.
