Эвапотранспирация, пренебрегая деталями, понимает гумус, при этом буквы А, В, I, О символизируют соответственно общеутвердительное, общеотрицательное, частноутвердительное и частноотрицательное суждения. Символизм психологически переоткладывает электронный символизм.
