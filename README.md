Согласно классификации М.Вебера,  растворение ненаблюдаемо заряжает заснеженный резонатор, при этом разрешен провоз 3 бутылок крепких спиртных напитков, 2 бутылок вина; 1 л духов в откупоренных флаконах, 2 л одеколона в откупоренных флаконах.
