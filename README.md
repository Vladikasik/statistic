Изостазия катастрофично восстанавливает полимерный мир  - все дальнейшее далеко выходит за рамки текущего исследования и не будет здесь рассматриваться. Англо-американский тип политической культуры, в первом приближении, нейтрализует квазар, в итоге приходим к логическому противоречию.</p></div>
