перейти к поступательно перемещающейся системе координат, чем и характеризуется флюгель-горн, причём сам Тредиаковский свои стихи мыслил как “стихотворное дополнение” к книге Тальмана.</p><p>Арифметическая прогрессия нетривиальна.
