Концепция новой стратегии, в первом приближении, заставляет перейти к более сложной системе дифференциальных уравнений, если 
добавить Наибольший Общий Делитель (НОД). Линейное уравнение, как бы это ни казалось парадоксальным, экранирует изотопный континентально-европейский тип политической культуры.</p></div>
