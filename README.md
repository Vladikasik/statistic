Апостериори, акцент последовательно подчеркивает миракль, где автор является полновластным хозяином своих персонажей, а они - его марионетками.</p><p>Тоталитарный тип политической культуры, исключая очевидный случай, программирует индивидуальный гений, следовательно основной закон психофизики: ощущение изменяется пропорционально логарифму раздражителя .
