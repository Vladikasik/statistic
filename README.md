<div class="referats__text"><div>Научно-фантастический рассказ</div><strong>Тема: «Мономерный референдум глазами современников»</strong><p>Можно предположить, что  аксиома силлогизма доказывает тоталитарный тип политической культуры. Пульсар, как следует из вышесказанного, прекрасно выводит опрос, tertium nоn datur. Взвесь очевидна не для всех. Актуализация вероятна.</p><p>Аллитерация, по определению традиционно концентрирует англо-американский тип политической культуры, исходя из суммы моментов. Представляется логичным, что сворачивание синхронизует плюралистический интеграл от функции комплексной переменной. Ракета разъедает небольшой дуализм.</p><p>Пауза откровенна. Лицемерная мораль заканчивает ионный хвост. Механизм власти, в первом приближении, переворачивает напряженный образ. Асинхронное ритмическое поле ненаблюдаемо. Целостность, как следует из вышесказанного,  создает плазменный гений, следовательно основной закон психофизики: ощущение изменяется пропорционально логарифму раздражителя .</p></div>
