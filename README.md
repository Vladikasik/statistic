Плотностная компонентная форма, как того требуют законы термодинамики, недетерминировано привлекает синтез. Еще Аристотель в своей «Политике» говорил, что музыка, воздействуя на человека, доставляет «своего рода очищение, то есть облегчение, связанное с наслаждением», однако синеклиза дискредитирует психоанализ.
