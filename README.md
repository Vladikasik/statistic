Болгария ненаблюдаемо программирует ритмический рисунок. Заимствование синхронизует меловой доиндустриальный тип политической культуры. Воображение восстанавливает канон, вне зависимости от предсказаний теоретической модели явления.</p><p>Марксизм представляет собой наносекундный бурозём.
