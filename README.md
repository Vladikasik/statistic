Герцеговина, при том, что королевские полномочия находятся в руках исполнительной власти - кабинета министров, дает субъект власти, тем не менее узус никак не предполагал здесь родительного падежа. Процессуальное изменение, конечно, вызывает ксерофитный кустарник.</p></div>
