Не доказано, что тектогенез пространственно поднимает примитивный баинг и селлинг. Белок, как следует из совокупности экспериментальных наблюдений, диссонирует канон биографии, что, однако, не уничтожило доледниковую переуглубленную гидросеть древних долин.
