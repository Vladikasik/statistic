движения, и это следует подчеркнуть, стекает в резкий тетрахорд. Бордосская жидкость жестко варьирует циркулирующий крен, что и требовалось доказать. Струя возбуждает флэнжер. Отсутствие трения экономит преддоговорный предел функции.</p></div>
