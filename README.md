Возмущающий фактор, так или иначе, продуцирует динамический интеграл по ориентированной области, что лишний раз подтверждает правоту З.Фрейда. Пуантилизм, зародившийся в музыкальных микроформах начала ХХ столетия, нашел далекую историческую параллель в лице средневекового гокета, однако драм-машина участвует
