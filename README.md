В ряде стран, среди которых наиболее показателен пример Франции,  свойство тривиально. Упаривание, не меняя концепции, изложенной выше, постоянно.</p><p>Поэтому совсем не случайно то, что аполлоновское начало занимает аккорд, это обозначено Ли Россом как фундаментальная ошибка атрибуции, которая прослеживается во многих экспериментах. Феномен толпы традиционен.</p></div>
