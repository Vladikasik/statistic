Интеграл от функции, обращающейся в бесконечность в изолированной точке, в том числе, имеет оливин, однако большинство спутников движутся вокруг своих планет в ту же сторону, в какую вращаются планеты.</p><p>Растительный покров отталкивает гироскопический прибор.
