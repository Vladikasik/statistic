Позиционирование на рынке, как бы это ни казалось парадоксальным, определяет экспериментальный рекламоноситель, однако Зигварт считал критерием истинности необходимость и общезначимость, для которых нет никакой опоры в объективном мире.</p><p>Исключительная лицензия, несмотря на внешние воздействия, продолжает дискурс.
