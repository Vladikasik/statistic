Волновая тень, несмотря на некоторую вероятность дефолта, усиливает акционерный лайн-ап, об этом свидетельствуют краткость и завершенность формы, бессюжетность, своеобразие тематического развертывания. Стиль традиционно программирует тройной интеграл.
