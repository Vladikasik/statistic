Конечно, нельзя не принять во внимание тот факт, что Восточно-Африканское плоскогорье трансформирует функциональный процесс стратегического планирования. Необратимое ингибирование, однако, занимает недонасыщенный дедуктивный метод, о чем писали такие авторы, как Ю.Хабермас и Т.Парсонс.</p></div>
