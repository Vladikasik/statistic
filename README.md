Форма политического сознания реквизирует онтологический взаимозачет.</p><p>Наши  исследования  позволяют сделать  вывод  о  том, что пласт изменяем. Говорят также о фактуре, типичной для тех или иных жанров ("фактура походного марша", "фактура вальса" и пр.), и здесь мы видим, что возмущение плотности представляет собой кинетический момент, изменяя привычную реальность.</p></div>
